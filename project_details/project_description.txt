Our cohort designed and implemented a clock-driven robot control processor in Verilog, simulated using Icarus Verilog. The system is a fully synchronous, opcode-based digital processor that interprets 4-bit commands and updates internal state registers on each clock cycle. These registers store the robot’s speed, heading, X and Y position, LED state and color, firing signal, and weapon type.
The design is built using modular digital components, including D flip-flops for state storage, a 4-to-16 decoder for opcode interpretation, multiplexers and a demultiplexer for control flow, and arithmetic units such as adders and two’s complement blocks for state updates. Movement is implemented through move forward and move backward operations, where position updates depend on the robot’s current heading and speed. Additional opcodes control speed adjustments, turning, LED behavior, firing actions, and weapon selection. Overall, the project demonstrates a modular datapath and control architecture similar to a simplified processor, integrating multiple subcomponents into a cohesive digital control system.
